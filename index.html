Academic Pages is a ready-to-fork GitHub Pages template for academic personal websites

This is the front page of a website built with GitHub Pages. You can fork this starter, modify the configuration and Markdown files, add your own PDFs and other content, and have your site live for free—no ads!

Read the Guide See Portfolio

A data-driven personal website

Like many Jekyll-based templates, this approach separates content from form. Write in Markdown; templates transform your posts, pages, and metadata into HTML. Each commit you push to your public repo triggers GitHub Pages to rebuild and deploy your site.

Tip: keep your Markdown and assets under version control so you can safely roll back changes at any time.

Popular tools

  • Mermaid for diagrams
  • Plotly for plotting
